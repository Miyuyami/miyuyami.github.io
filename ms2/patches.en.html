Korean MapleStory 2 Patches

You can use a simple "Game Starter" that I made to help with starting the game without having to go through patching (you still need to patch when a new patch is out!!!)

This helps so that the nexon launcher won't start and replace the patched files. You can find it on my github by clicking here.

It's very easy to setup, make a shortcut of the GameStarter.exe then right click the shortcut and select Properties. Then under "Target" add at the end " 106498" (without quotes and note the space before the number).

So if the target is "C:\My Games\GameStarter.exe" you must edit it to "C:\My Games\GameStarter.exe" 106498. All you have to do now is run it from the shortcut.

As for how to use... open it and select the "Launch" option to start the game and skipping the file checking, if an error shows up or you know that there's been an update then choose the "Patch" option (make sure you put the backup "Xml" files so that they won't get redownloaded). Now wait until a new patch is available here (check "Last Update") then replace the files, now in the "game stater" choose launch again.

Go to "MapleStory2\Config\GameOption.xml" and in the line with "language" change it as the table below.

If you get boxes instead of text, you might need to change the fonts by editing a config file at "MapleStory2\Data\Resource\gfx\fontconfig.txt", you might need the NexonGameStarter from above.

Chinese MapleStory 2 Patches

The English patch is based off the Global client (GMS2) and the Korean patch is from the Korean client (KMS2).

The "No Names version" has the items and achievements/trophies names are left in chinese so that you can safely link them in chat without anyone finding you suspicious (hopefully).

Make sure you backup your original Xml.m2h and Xml.m2d (or just Xml if you cannot see extensions)

You'll also need to remove the patch and place the original files back to update your game.
