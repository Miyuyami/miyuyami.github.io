Chinese MapleStory 2 Patches

The English patch is based off the Global client (GMS2) and the Korean patch is from the Korean client (KMS2).

The "No Names version" has the items and achievements/trophies names are left in chinese so that you can safely link them in chat without anyone finding you suspicious (hopefully).

Korean MapleStory 2 Patches

Same as the Chinese patches but for Korea...

You can use a simple "Game Starter" that I made to help with starting the game without having to go through patching (you still need to patch when a new patch is out!!!)

This helps so that the nexon launcher won't start and replace the patched files. You can find it on my github by clicking here.

It's very easy to setup, make a shortcut of the GameStarter.exe then right click the shortcut and select Properties. Then under "Target" add at the end " 106498" (without quotes and note the space before the number).

So if the target is "C:\My Games\GameStarter.exe" you must edit it to "C:\My Games\GameStarter.exe" 106498. All you have to do now is run it from the shortcut.

As for how to use... open it and select the "Launch" option to start the game and skipping the file checking, if an error shows up or you know that there's been an update then choose the "Patch" option (make sure you put the backup "Xml" files so that they won't get redownloaded). Now wait until a new patch is available here (check "Last Update") then replace the files, now in the "game stater" choose launch again.

Global MapleStory 2 Patches

Same as the other patches but for global client...

You can change to one of the available language from options after you create a character. (ESC -> Options -> Languages -> Language box -> Scroll to the language that you want)

If you get some error with "keyname.xml", go to "YourClientFolder\Config\GameOption.xml" and in the last few rows change "languageIndex" to "2", which is the default English.

Fonts only version adds the same languages in options but the game stays completely in english. All that changes will be the fonts used for it's respective language. This is useful for reading and writing in those respective languages while keeping the game text completely in English.

Global MapleStory 2 Sound Packs

To use different voice files by just copying the sound files off another region. You do not copy all the sound files because you do NOT want to overwrite anything, you just copy the files that contain "kor" or "cn" in their file names, depending on the respective region.

After placing the sound files near your existing global files, which should be located in "YourClientFolder\Data\Sound", you will have to change a config file, which is located in "YourClientFolder\Config\GameOption.xml", somewhere in the last few rows it says "voiceIndex="2", 2 stands for English, change it to 0, which stands for Korean. For Chinese you have to change it to 3. For Japanese you have to change it to 1.

If you get some error with "keyname.xml", go to "YourClientFolder\Config\GameOption.xml" and in the last few rows change "languageIndex" to "2", which means English.

I haven't tested them much but they seem to work on the sounds that are also present on the respective versions, including emotes, but something like the GMS2 CBT emote will NOT work because it does not exist in the Korean sound files.
