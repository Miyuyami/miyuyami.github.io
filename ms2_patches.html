You can use a simple "Game Starter" that I made to help with starting the game without having to go through patching (you still need to patch when a new patch is out!!!)

This helps so that the nexon launcher won't start and replace the patched files. You can find it on my github by clicking here.

It's very easy to setup, make a shortcut of the GameStarter.exe then right click the shortcut and select Properties. Then under "Target" add at the end " 106498" (without quotes and note the space before the number).

So if the target is "C:\Games\GameStarter.exe" you must edit it to "C:\Games\GameStarter.exe 106498". All you have to do now is run it from the shortcut.


As for how to use... open it and select the "Launch" option to start the game and skipping the file checking, if an error shows up or you know that there's been an update then choose the "Patch" option (make sure you put the backup "Xml" files so that they won't get redownloaded). Now wait until a new patch is available here (check "Last Update") then replace the files, now in the "game stater" choose launch again.

You can change to Korean or Chinese from options after you create a character. (ESC -> Options -> Languages -> Language box -> Scroll to Korean or Chinese)

If you get some error with "keyname.xml", go to "YourClientFolder\Config\GameOption.xml" and in the last few rows change "languageIndex" to "2", which means English.
