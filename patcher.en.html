Description

Soulworker Patcher was made to easily download and patch latest translations into your Soulworker Client using the files from your client. This makes it compatible with almost any version, even if the translation is outdated. It is still recommended to always get the latest translations.

The translations are downloaded from SWHQ Translation team and the original files are backed up, they will be restored after the client closes.

Soulworker Patcher requires the game to be installed before using it. Both Japanese version and Korean versions are supported. If you have both installed, you can change the "Game Location" if you want to access the other region.

Functions

  • Backs up the original files and replaces them with the downloaded translations.
  • Auto-updates the Soulworker client when it's outdated. (JP region only)
  • Unicode codepage fix to support unicode languages excluding the server region's language
  • Direct login for faster startup. (JP region only, for KR region it only opens Stove)

FAQ (Common Problems)

  • For any error when installing or running the patcher, try these fixes first

    • Make sure you have the requirements to install or manually install Microsoft .NET Framework 4.6.2
    • Add "%localappdata%\Apps\2.0" to your anti-virus exception list
    • Run the setup.exe as administrator
    • Reset Internet Explorer settings, Internet Explorer advanced settings and Internet Explorer security zones. Those can be found in Control Panel -> Internet Options

  • "Application cannot be started. Contact the application vendor." error on startup

    • uninstall the Soulworker Patcher from Control Panel/Programs and Features
    • go to "%localappdata%\Apps\2.0" then 2 more folders with random letters
    • you should see a lot of folders with long names, delete the ones starting with "swpa"
    • now go in "manifests" folder and do the same as above
    • then go back to "2.0" folder and do the same for "Data" folder
    • after all that, press "Windows Key + R" (Run box) and type (without quotes)
    • "rundll32 %SystemRoot%\system32\dfshim.dll CleanOnlineAppCache"
    • press enter
    • try to install Soulworker Patcher again

  • At server selection screen, the server name looks like some gibberish...

    It's a side effect of the unicode codepage fix. It will not affect anything else.

  • I get this error when starting the patcher

    • for some reason Soulworker Patcher couldn't gain access to the game folder
    • you should add the permissions manually, like this
    • go to your "soulworker" folder Properties on Security tab
    • then go to Edit, then Add
    • write the name of your windows user, press Check Names, then press OK
    • select the group that was created and check the boxes like this
      note: instead of "Miyu" it should be the name of your windows user
    • save the changes
    • run Soulworker Patcher again and it should work
